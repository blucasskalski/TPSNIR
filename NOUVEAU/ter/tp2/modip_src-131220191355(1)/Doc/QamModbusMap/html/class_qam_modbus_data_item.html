Description détaillée

Modélisation d'une donnée primaire ou d'un champ partiel de donnée 16 bits.

La classe QamModbusDataItem est à usage privé de QamModbusData. Elle encapsule les caractéristiques d'une donnée : nom, commentaire, masque, format et valeur.
Une donnée primaire ou le premier mot d'une donnée composée à toujours un masque égal à $FFFF. Un champ partiel d'une donnée primaire (dit donnée secondaire) à un masque non nul différent de $FFFF. Un mot suivant d'une donnée composée à un masque égal à $0000.
