Description détaillée

Thread de gestion d'une connexion TCP.

La classe QamTcpConnection prend en charge l'établissement et la gestion d'une connexion TCP avec un client. Elle utilise comme serveur pour répondre aux requêtes l'objet dérivé de QamAbstractServer (classe abstraite) reçu lors de la construction. Le serveur est aussi destinataire de différents messages de suivi de connexion et/ou d'erreur.
