Description détaillée

Serveur TCP multithread.

La classe QamTcpServer permet la mise en oeuvre d'un serveur TCP/IP multi-clients. La logique applicative de réponse aux requêtes est assurée par un objet répondant au modèle de développement spécifié par la classe QamAbstractServer (voir documentation de QamAbstractServer).
